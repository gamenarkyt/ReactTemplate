# Base React Template

## What it use:

### Base:

- React
- React-router-dom
- React-query
- axios
- useHooks

### Additional:

- useHooks library
- import alias (configure your vscode to non-relative import)
- svgr plugin for import svg image
- reset css file (test version)
- husky pre-commit hook (eslist and prettier analyze and fix before commit)
